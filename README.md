# 电工模拟器

面向电工初学者的网页电路学习工具：拖拽电气元件、自由接线、操作开关按钮，通过模拟运行理解低压电工电路图中的通断关系、接触器动作逻辑和常见接线错误。

第一版聚焦三个核心练习：**单灯单控、接触器点动、接触器自锁**。

## 目录结构

```
docs/    产品与设计文档
  电工模拟器网页产品需求文档 PRD.md
  电路连通性判定规约.md          模拟引擎的实现级规格
engine/  纯逻辑模拟引擎（TypeScript，零依赖，含测试）
web/     网页应用（Vite + React + React Flow，复用 engine）
```

## 快速开始

跑网页应用：

```bash
cd web
npm install
npm run dev        # 打开 http://localhost:5173/
```

跑引擎测试（需 Node ≥ 22.6）：

```bash
cd engine
npm test
```

## 设计思路

- **引擎优先**：先把"判断电路通不通、错在哪"的核心逻辑做成独立、有测试的纯 TypeScript（`engine/`），再套界面。浏览器跑的就是同一份逻辑。
- **状态模拟**：第一版只判断通/断、得电/失电、亮灭、转停、对错，不做精确电流电压计算。引擎规约见 `docs/电路连通性判定规约.md`。
- **写实图形 + 中文名**：元件用带光影的写实画法并随状态变化，配中文名牌，方便初学者看懂。
